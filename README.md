# 🔊 Sound Board - 250+ Sounds

A comprehensive, interactive sound board website with over 250 sounds organized into 4 major categories. Perfect for pranks, entertainment, relaxation, and music discovery!

## 🎯 Features

✨ **250+ High-Quality Sounds** organized into:
- 🏎️ **Engine Room** (150 sounds) - Cars, Motorcycles, F1, Top Fuel Dragsters, Rotors, Muscle Cars
- 🎧 **ASMR** (120 sounds) - Water, Cutting, Nature, Sleep, Cooking, Crafting
- 😂 **Meme Space** (120 sounds) - Classic Memes, Movie Quotes, Gaming, Comedy, Internet Culture, Meme Music
- 🎵 **Music** (120 sounds) - Trending, Rock, Hip-Hop, Electronic/EDM, Pop, Chill

🔍 **Search Functionality** - Instantly find any sound
🔊 **Master Volume Control** - Adjust all sound levels
🎨 **Beautiful UI** - Modern gradient design with smooth animations
📱 **Responsive Design** - Works on desktop, tablet, and mobile
⌨️ **Keyboard Shortcuts**:
  - `Space` - Play/Pause
  - `Escape` - Stop current sound
🎵 **Now Playing Display** - See what's currently playing

## 🚀 How to Use

1. **Open** `index.html` in your web browser
2. **Browse** different sound categories using the navigation tabs
3. **Click** any sound button to play it
4. **Search** for specific sounds using the search bar
5. **Adjust** volume with the master volume slider
6. **View** currently playing sound in the floating display
